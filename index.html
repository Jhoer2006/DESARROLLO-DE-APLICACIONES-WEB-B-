Inicio

Bienvenido a mi sitio web, desarrollado como parte de las actividades prácticas de la asignatura Desarrollo de Aplicaciones Web de la carrera Ingeniería en Tecnologías de la Información. Este proyecto tiene como finalidad poner en práctica los conocimientos básicos adquiridos sobre HTML5, CSS3 y Bootstrap mediante la creación de una página web organizada, funcional y adaptable a diferentes dispositivos.

Quiénes Somos

Este proyecto fue elaborado con fines académicos para reforzar los conocimientos adquiridos en la asignatura Desarrollo de Aplicaciones Web. Durante su desarrollo se aplicaron conceptos fundamentales relacionados con la estructura de una página web, el uso de etiquetas semánticas y la organización adecuada de la información. La página integra diferentes elementos como menús de navegación, imágenes, contenido multimedia y secciones informativas que permiten comprender de manera práctica cómo se construye un sitio web básico utilizando HTML5. Además, esta actividad contribuye al fortalecimiento de habilidades relacionadas con el diseño y desarrollo de páginas web.

Temas Aplicados en la Práctica

HTML5

Uso de etiquetas semánticas para estructurar correctamente una página web.

CSS3

Aplicación de estilos visuales para mejorar la apariencia del sitio web.

Bootstrap

Uso de componentes responsivos para adaptar el diseño a diferentes dispositivos.
