Inicio

Bienvenido a mi sitio web, desarrollado como parte de las actividades prácticas de la asignatura Desarrollo de Aplicaciones Web, de la carrera Ingeniería en Tecnologías de la Información. Este proyecto tiene como finalidad poner en práctica los conocimientos básicos adquiridos sobre HTML5, mediante la creación de una página web organizada y funcional. A lo largo de esta práctica se utilizaron diferentes elementos y etiquetas semánticas que permiten estructurar correctamente la información, así como incorporar imágenes, enlaces y contenido multimedia para mejorar la experiencia del usuario.

Quiénes Somos

Este proyecto fue elaborado con fines académicos para reforzar los conocimientos adquiridos en la asignatura Desarrollo de Aplicaciones Web. Durante su desarrollo se aplicaron conceptos fundamentales relacionados con la estructura de una página web, el uso de etiquetas semánticas y la organización adecuada de la información. La página integra diferentes elementos como menús de navegación, imágenes, contenido multimedia y secciones informativas que permiten comprender de manera práctica cómo se construye un sitio web básico utilizando HTML5. Además, esta actividad contribuye al fortalecimiento de habilidades relacionadas con el diseño y desarrollo de páginas web, permitiendo familiarizarse con herramientas y metodologías utilizadas en proyectos reales dentro del ámbito tecnológico.

Temas Aplicados en la Práctica

  • Uso de etiquetas semánticas de HTML5.
  • Organización y estructuración del contenido web.
  • Inserción de imágenes dentro de una página web.
  • Integración de videos mediante iframe.
  • Creación de enlaces internos y externos.
